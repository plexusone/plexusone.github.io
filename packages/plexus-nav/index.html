PlexusOne Navigation Component

This is a development page for testing the <plexus-nav> web component. The navigation bar is rendered above using Lit Web Components.

Test Cases

  • Click "Products" to open the mega menu
  • Click "Developers" or "Community" for dropdown menus
  • Press Escape to close any open menu
  • Resize the window to test mobile responsive behavior
  • On mobile, tap the hamburger menu icon
  • Test keyboard navigation (Tab, Enter, Escape)

Usage

<!-- ES Module -->
<script type="module">
  import '@plexusone/nav';
</script>
<plexus-nav></plexus-nav>

<!-- Standalone (IIFE) -->
<script src="https://plexusone.dev/js/plexus-nav.min.js"></script>
<plexus-nav></plexus-nav>

Configuration

<plexus-nav id="nav"></plexus-nav>
<script>
  document.getElementById('nav').config = {
    baseUrl: 'https://plexusone.dev',
    productsUrl: 'https://plexusone.dev/data/products.json',
    currentProduct: 'omnillm'
  };
</script>
